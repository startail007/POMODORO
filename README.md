# 番茄鐘
### 比賽網址
##### 參與前端修練精神時光屋比賽(https://challenge.thef2e.com/)
### 內容
##### 1.倒數分為工作跟休息的時間，可新增代辦事項，選擇要得代辦事項後啟動倒計時，會25分鐘倒數顏色為紅色，等時間倒會跳為藍色5分鐘休息時間
##### 2.在to-do list內，勾選已完成的代辦事項，如要刪除可按住拖曳到刪除處放開
##### 3.在analytics內看到一周內的時間
##### 4.在ringtones設置工作與休息時間完成後的提示音
